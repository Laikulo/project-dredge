# Project Dredge.
Dredge is an attempt to catalog vendor modifications to, and implementations of [Klipper](https://klipper3d.com).

## Dependencies
* make
* tar
* curl
* virt-tar-out (from libguestfs)
* podman
* unsquashfs
* rk2918\_tools (vendored)
* upfile (vendored)
